## Role

## Context

Я создаю персонального Telegram-бота для работы с голосовыми заметками.

Основной сценарий:

* Я отправляю голосовое сообщение или аудиофайл в Telegram
* Бот отправляет аудио в DeepGram API
* Получает транскрипцию (на русском языке, с пунктуацией и нормализацией)
* Возвращает мне текст в Telegram

Дополнительно:

* По кнопке я могу сохранить:

  * только текст
  * аудио + текст
* Сохранение происходит в Google Drive
* Формат сохранения: `.md`
* Файлы группируются по дате (папки YYYY-MM-DD)

Важно:

* По умолчанию ничего не сохраняется
* Только по явному действию пользователя

## Task

Сгенерируй полный минимальный, но расширяемый проект Telegram-бота, который:

### 1. Обрабатывает входящие сообщения:

* voice messages
* audio files

### 2. Выполняет:

* скачивание файла из Telegram
* отправку файла в DeepGram API
* получение транскрипции
* отправку текста пользователю

### 3. Реализует UI через inline-кнопки:

После транскрипции:

* кнопка: "Сохранить текст"
* кнопка: "Сохранить аудио + текст"

### 4. Реализует Google Drive:

* авторизация через service account
* создание папки по текущей дате (если нет)
* сохранение `.md` файла

Формат `.md` файла:

```md
# Voice Note

Дата: YYYY-MM-DD HH:MM

## Текст
<транскрипция>

## Файл
(если есть аудио)
```
