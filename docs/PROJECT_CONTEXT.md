# Project Context

## Цель
Telegram-бот для транскрибации голосовых сообщений через DeepGram.

## Стек
- Python
- python-telegram-bot (async)
- DeepGram API
- Google Drive API

## Поведение
- Принимает voice/audio
- Возвращает текст (короткие сообщения)
- Для длинных сообщений показывает превью и кнопку "Показать полностью"
- Кнопки сохранения: текст (TXT/MD), аудио + текст (TXT/MD)
- После сохранения кнопки остаются для повторного выбора

## Ограничения
- простой код
- без сложных архитектур
- всё должно быть понятно

## Правила
- писать комментарии в коде
- не усложнять
- использовать logging вместо print
